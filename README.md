📌 Project Overview

This project focuses on analyzing Electric Vehicle (EV) data to understand market trends, vehicle distribution, and technological progress. The analysis includes both:

Battery Electric Vehicles (BEVs)
Plug-in Hybrid Electric Vehicles (PHEVs)

The goal is to derive meaningful insights using key performance indicators (KPIs) and visualize them through an interactive dashboard.

🎯 Objectives

The main objectives of this project are:

Analyze the total number of electric vehicles
Evaluate the average electric range
Compare BEVs and PHEVs distribution
Understand market share and dominance
📊 Key Performance Indicators (KPIs)
1. 🚗 Total Vehicles
Calculates the total number of electric vehicles (BEVs + PHEVs)
Helps understand the overall EV market size
2. 🔋 Average Electric Range
Computes the average electric range of vehicles
Indicates improvements in EV technology and efficiency
3. ⚡ Total BEV Vehicles & Percentage
Total number of Battery Electric Vehicles (BEVs)
Percentage of BEVs relative to total EVs

Formula:

BEV % = (Total BEVs / Total EVs) × 100
4. 🔌 Total PHEV Vehicles & Percentage
Total number of Plug-in Hybrid Electric Vehicles (PHEVs)
Percentage of PHEVs relative to total EVs

Formula:

PHEV % = (Total PHEVs / Total EVs) × 100
